# Phil's Phones Data Analysis Project


This Jupyter notebook project explores the mobile phone industry and provides insights into various aspects of the business, from market analysis to predictive modeling and data exploration.

## Project Overview

Explore the mobile phone industry, understand market dynamics, and develop data-driven insights for Phil's Phones.

The project is divided into several sections, each addressing specific aspects of the business:

- **Motivation**: Understand the importance of the mobile phone industry and the driving factors.
- **Domain Knowledge**: Explore clock speed in mobile phones and its relevance.
- **Decision-making Framework**: Discuss the utility of predictive models in pricing and decision-making.
- **Data Analytics Lifecycle**: Create a framework tailored to the mobile phone industry.
- **Pre-processing**: Prepare the dataset for analysis.
- **Data Exploration**: Visualize data using 3D scatterplots and additional exploratory analysis.
- **Training a Neural Network**: Predict battery power for high-priced phones.
- **Classification**: Build a model to classify phone price categories.
- **Outlier Detection**: Train an autoencoder to identify outliers.
- **Dimensionality Reduction**: Condense data for visualization.

## Context

Phil's Phones, a leading mobile phone retailer, aims to manufacture smartphones. This project addresses critical questions, conducts comprehensive data analysis, and provides valuable insights for informed decision-making.

## Dataset

The project uses a dataset with 2000 rows and 15 columns, representing various specifications of mobile phones, including battery power, camera specifications, memory, and more.

## Conclusion

This analysis equips Phil's Phones with valuable insights into the mobile phone industry, from pricing strategies to data-driven decision-making. For detailed analyses, please refer to the Jupyter notebook.
